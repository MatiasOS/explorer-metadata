# Yeenus Token (YEENUS)

Yeenus is a test ERC20 token with 8 decimals, designed for development and testing purposes.

## Overview

- **Symbol**: YEENUS
- **Decimals**: 8
- **Network**: Localhost (31337)

## Purpose

Yeenus tokens provide developers with a test token that has non-standard decimals (8 instead of 18), useful for:

- Testing applications with different decimal configurations
- Simulating USDT-like tokens (which also use 8 decimals)
- Validating decimal handling in DeFi protocols

## How to Get Tokens

Yeenus tokens can be obtained for free from the token faucet by sending a 0 ETH transaction to the contract address.
