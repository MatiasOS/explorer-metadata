# Weenus Token (WEENUS)

Weenus is a test ERC20 token designed for development and testing purposes on Ethereum and EVM-compatible networks.

## Overview

- **Symbol**: WEENUS
- **Decimals**: 18
- **Network**: Localhost (31337)

## Purpose

Weenus tokens are freely available test tokens that developers can use to:

- Test token transfers and approvals
- Develop and debug DeFi applications
- Test wallet integrations
- Experiment with smart contract interactions

## How to Get Tokens

Weenus tokens can be obtained for free from the Weenus Token Faucet. Simply send a 0 ETH transaction to the token contract address to receive test tokens.

## Technical Details

Weenus implements the standard ERC20 interface with additional faucet functionality. Anyone can request tokens by calling the `drip()` function or sending a zero-value transaction to the contract.
